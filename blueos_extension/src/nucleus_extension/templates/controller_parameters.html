The Nucleus extension relies on sending velocity data into the BlueROV in order for the BlueROV to able to use its "Position Hold" feature.

In order for the BlueROV to retrieve this velocity data, certain paramaters in the BlueROV controller has to be changed.

These parameters and their required values are as follows:

  • AHRS_EKF_TYPE = 3
  • EK2_ENABLE = 0
  • EK3_ENABLE = 1
  • VISO_TYPE = 1
  • GPS_TYPE = 0
  • EK3_SRC1_POSXY = 6
  • EK3_SRC1_VELXY = 6
  • EK3_SRC1_POSZ = 1
  • SERIAL0_PROTOCOL = 2

N.B. These changes may affect the behaviour of the BlueROV!

N.B. Neither this extension nor the BlueROV will remember the default values of these parameters.

It is recommended that users write down the original values before making changes to these parameters, in case they wish to chenge them back later
